## 简介
- 模仿豆瓣写的读书网站
- 使用python的django框架

## 下载运行
1. '''git clone git@github.com:YJGit/reading_website.git'''  
2. 进入reading_website目录，打开命令行，运行'''python manage.py runserver 0.0.0.0:8000'''
3. 打开浏览器，输入http://localhost:8000/，即可看到网站
